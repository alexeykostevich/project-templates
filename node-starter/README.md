# Node.js Starter

A starter for a Node.js application.

## Prerequisites

- Install [Node.js](https://nodejs.org/)

## Quickstart

```sh
# Install the dependencies.
npm ci

# Start the app in the dev mode.
npm start:dev

# Build the app.
npm build

# Start the app.
npm start
```

> **TIP**: The project is already set up for debugging in VS Code.
